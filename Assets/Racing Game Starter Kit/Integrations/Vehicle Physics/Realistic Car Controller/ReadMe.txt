RCC v3.2b2 Integration

--- General Setup ---

1. Set Main Controller Type to "Custom" in RCCSettings.



--- Player Vehicle Setup ---

1. Set tag to "Player"
2. Add the RaceCompetitor.cs script to the vehicle
3. Add the RCCPlayerInput.cs script to the vehicle
4. Add camera target via the Vehicle Setup window "Extra" tab



--- AI Vehicle Setup ---

1. Set tag to "Untagged"
2. Add the RaceCompetitor.cs script to the vehicle
3. Add the AIController.cs script to the vehicle
4. Add the RCCAIInput.cs script to the vehicle
